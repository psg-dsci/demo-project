Setup Checklist

GitHub Secrets required

Add these four secrets in your repo → Settings → Secrets and variables → Actions. Click each to mark done.

EC2_HOST
Your EC2 public IP or DNS hostname !!(e.g. 54.123.45.67)
EC2_USER
ec2-user for Amazon Linux, ubuntu for Ubuntu AMIs
EC2_SSH_KEY
Full contents of your .pem private key file including headers
REPO_URL
https://github.com/YOUR_USERNAME/YOUR_REPO.git
